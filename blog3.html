Blog Post

This week helped me better understand the process of creating and running an online business, especially when it comes to building a website and choosing where to host it. One of the biggest things I thought about was whether it would be better to use a website builder like Weebly or Wix, or to build a site completely from scratch using HTML and CSS. Website builders seemed easier because they provide templates, hosting, and drag-and-drop tools, which can help businesses launch websites quickly. At the same time, coding with HTML and CSS allows for much more customization and control over the design and functionality of a website.

In the end, I decided that building the website myself with HTML and CSS was the best choice for me. Even though it takes more time and technical knowledge, I liked having full control over the website instead of relying on templates. I also compared hosting options like GitHub, Bluehost, and GoDaddy. After looking at the advantages of each option, I decided GitHub worked best for my project because it allows me to host my website for free while also helping me manage and improve my code. Overall, this week showed me that creating a successful online business is not only about selling products, but also about choosing the right platform and making a website that fits the goals of the business.
